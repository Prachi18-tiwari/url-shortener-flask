# URL Shortener using Flask

A simple URL Shortener web application built using Python and Flask.
It converts long URLs into short links and redirects users to the original URL.

## Features
- Shorten long URLs
- Redirect short URLs
- Flask backend with HTML & CSS frontend
- SQLite database support

## Tech Stack
- Python
- Flask
- SQLite
- HTML
- CSS

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install flask flask_sqlalchemy
3. Run the app:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/
